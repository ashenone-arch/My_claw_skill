# 投研素材分析模板 · 9段结构

> 本模板用于 AI 生成公司深度投研素材，非客户路演报告。
> 素材包帮分析师快速建框架、拿数据、知道缺什么。
>
> **叙事统一模式**：每段遵循「数据锚点 → 分析展开 → 段内综合判断」三段式。
> 详细分析框架选择指南见 `framework-toolkit.md`。

---

## 证据等级表

| 标签 | 含义 | 示例 |
|------|------|------|
| F1 | 公开可验证事实 | 交易所披露 |
| F2 | 财报/金融数据库/公告 | 年报、iFinD 数据 |
| M1 | 市场一致预期 + 管理层公开发言 | 卖方预测、业绩会发言 |
| C1 | 合理推演 | 需附推演链路 |
| H1 | 待核验假设 | 必须标"未经核验" |

**约束**：没有证据等级 = 不能写。关键结论必须有 F1/F2 支撑。管理层公开发言 = M1，不是 F2。

### 段1 · 公司定位

**所需数据**：`get_stock_summary` / `get_stock_info` / `search_stocks`（可比公司）

**核心假设**：我们假设 [公司的核心壁垒在于X，差异化定位体现在Y]。若 [X被证伪或Y的同质化加剧]，则本段的定位结论需要修正。

**写作模板**：

```
### 产业链位置
公司在 [行业] 产业链中处于 [上游/中游/下游] [具体环节]（F1-[来源]）
- 上游：[原材料/技术/设备供应商]，议价地位：[强/中/弱]
- 下游：[客户/应用场景]，客户集中度：[高/中/低]（F2-iFinD 前5大客户占比）

### 产品/业务矩阵
| 业务线 | 收入占比 | 行业地位 | 所处生命周期 | 竞争格局 |
|--------|---------|---------|------------|---------|
| [业务A] | XX% | [龙头/跟随/利基] | [导入/成长/成熟/衰退] | [集中/分散] |
| [业务B] | XX% | [龙头/跟随/利基] | [导入/成长/成熟/衰退] | [集中/分散] |

### 差异化定位 vs 可比（≥3家）
| 维度 | 本公司 | 可比A | 可比B | 可比C |
|------|-------|-------|-------|-------|
| 技术路线 | | | | |
| 客户结构 | | | | |
| 区域布局 | | | | |
| 盈利模式 | | | | |

> 可比公司选择理由：A（同产品/相似规模），B（同客户群/不同产品），C（同行业/不同细分）。详细的定量对比见段7。
```

**写作规范**：
- 证据：F2 优先，C1 需附推演链路
- 数据溯源三要素：证据等级 + 来源 + 数据时间
- 可比公司选择理由必须写明，不能随意选

**段内综合判断**：

```
### 段1 综合判断
**[一句话：公司在产业链中处于X位置，核心差异化在于Y，此举定了公司的天花板/竞争格局]**
→ 数据支撑：[引用本段产业链位置+产品矩阵的关键 F2 数据]
→ 对投资判断的影响：此定位构成整体投资论点的 [基础支撑/主要风险来源]。
```

> **衔接段2**：本段识别的业务线结构将直接用于段2 的详细财务拆解。

### 段2 · 业务拆分

**所需数据**：`get_stock_financials`（主营构成拆解）

**核心假设**：我们假设 [业务A 是收入增长的主要驱动力，业务B 的毛利率在行业中处于领先水平]。若 [A的YoY增速降至X%以下 / B的毛利率相对优势消失]，则增长逻辑需要修正。

**写作模板**：

```
| 业务线 | 收入（亿） | 占比 | YoY | 毛利率 | 毛利贡献 | 证据 |
|--------|-----------|------|-----|--------|---------|------|
| [业务A] | XX | XX% | +X% | XX% | XX亿 | F2 |
| [业务B] | XX | XX% | +X% | XX% | XX亿 | F2 |
| [业务C] | XX | XX% | +X% | XX% | XX亿 | F2 |
| **合计** | **XX** | **100%** | **+X%** | **XX%** | **XX亿** | F2 |

⚠️ 数据缺失说明（如有）
```

**写作规范**：
- 元→亿，保留2位小数
- 缺失数据标 `[需补]`，段9 列出获取路径
- 新增"毛利贡献"列——识别哪些业务是利润核心

**段内综合判断**：

```
### 段2 综合判断
**[一句话：业务A贡献了主要增长和利润，但高度依赖该业务是风险；业务B毛利率领先但占比有限]**
→ 数据支撑：[引用收入和毛利贡献占比的核心对比数据]
→ 对投资判断的影响：收入结构决定了公司的 [增长质量/周期敏感性/利润弹性]。若业务A增速放缓且无新业务接棒，整体增长叙事将动摇。
```

> **衔接段3**：本段识别的增长驱动力（量/价/结构/客户）将在段3 中展开为四维分析。

---

### 段3 · 收入驱动

**所需数据**：`get_stock_financials` + `get_edb_data` + `get_stock_shareholders`

**核心假设**：我们假设 [收入增长主要由X因子驱动（量/价/结构/客户）]，行业宏观数据 [支撑/不支撑] 当前增速的可持续性。若 [X因子出现逆转信号]，则收入增速预测需要下调。

**写作模板**：

```
按"量/价/结构/客户"四维拆解：

- **[量]**：[结论]（证据等级-来源-数据时间）
  → 行业宏观佐证：[引用 get_edb_data 的产销量/价格指数数据]

- **[价]**：[结论]（证据等级-来源-数据时间）
  → 行业宏观佐证：[ASP趋势、PPI等]

- **[结构]**：[结论]（证据等级-来源-数据时间）
  → 产品组合升级路径：[高毛利产品占比变化趋势]

- **[客户]**：[结论]（证据等级-来源-数据时间）
  → 线索来源：前5大客户集中度、股东结构中ETF持仓变化
```

**写作规范**：
- 禁止把管理层公开发言当 F2 事实（发言 = M1）
- 每条结论带数据溯源三要素
- 量/价维度必须有行业宏观数据（`get_edb_data`）的佐证

**段内综合判断**：

```
### 段3 综合判断
**[一句话：收入增长由X因子驱动，可持续性为高/中/低]**
→ 数据支撑：[引用量/价/结构/客户中的核心 F2 数据及行业宏观佐证]
→ 对投资判断的影响：收入驱动因子的性质（量驱动 vs 价驱动 vs 结构升级）决定了增长的 [可预测性/波动率/天花板]。若驱动因子切换（如从量驱动转为价驱动），估值逻辑可能变化。
```

> **衔接段4**：本段的收入驱动分析为段4 判断利润增长弹性提供收入端锚点——收入增长能否转化为利润增长，取决于各驱动因子对利润率的影响。

### 段4 · 利润驱动

**所需数据**：`get_stock_financials`（利润表 + FY1-FY3 预测指标）

**核心假设**：我们假设 [利润增长弹性主要来自X（毛利率改善/费用率压缩/规模效应）]，一致预期 [存在/不存在] 显著下调风险。若 [X因子弱化 / 一致预期连续2次下修]，则盈利预测需重新评估。

**写作模板**：

```
### 盈利能力
- 毛利率：[数字]（F2），同比 [改善/恶化] XXbp
- 净利率：[数字]（F2），同比 [改善/恶化] XXbp
- ROE：[数字]，同比+/-XXbp（F2）

### DuPont 拆解（框架详见 framework-toolkit.md）
ROE = 净利率 × 资产周转率 × 权益乘数

| 因子 | 当前值 | 同比变化 | 近3年均值 | 驱动判断 |
|------|-------|---------|---------|---------|
| 净利率 | XX% | +/-XXbp | XX% | [改善/恶化/稳定] |
| 资产周转率 | XXx | +/-XX | XXx | [加快/放缓/稳定] |
| 权益乘数 | XXx | +/-XX | XXx | [加杠杆/去杠杆/稳定] |
| **ROE** | **XX%** | **+/-XXbp** | **XX%** | **主要由[X因子]驱动** |

→ ROE 变化的驱动力可持续性：[高/中/低]（C1-依据：[XX]）
→ 与可比公司模式差异：[公司A依赖高净利率，公司B依赖高杠杆...]

### 弹性分析
- 若[变量]变动X%，则利润变动约XX亿（C1-基于历史数据回归）

### 一致预期
| 机构 | FY1 收入(亿) | FY1 净利润(亿) | FY2 净利润(亿) | FY1 EPS | 预测日期 |
|------|------------|--------------|--------------|--------|---------|
| [机构A] | XX | XX | XX | XX | YYYY-MM-DD |
| [机构B] | XX | XX | XX | XX | YYYY-MM-DD |
| **均值** | **XX** | **XX** | **XX** | **XX** | — |
| 覆盖家数 | — | X家 | X家 | X家 | — |

> 预测日期超过3个月 → 标注"可能失准"。分析师覆盖 < 3家 → 标注"覆盖不足"。
```

**写作规范**：
- ROE/净利率必须有具体数字；必须做 DuPont 三因子拆解
- 一致预期表格增加收入/EPS/覆盖家数维度
- 预测日期超过3个月需提示"可能失准"

**段内综合判断**：

```
### 段4 综合判断
**[一句话：利润弹性来自X，ROE由Y因子驱动（可持续性Z），一致预期偏乐观/保守/合理]**
→ 数据支撑：[引用 DuPont 拆解+一致预期中的核心数据]
→ 对投资判断的影响：利润驱动的可持续性直接决定当前估值的合理性。若 ROE 主要由不可持续的杠杆驱动，或一致预期存在系统性高估，整体投资论点需要下调置信度。
```

> **衔接段5**：本段的盈利能力分析为段5 评估竞争力护城河提供财务基础——高 ROE 是否为护城河的结果？能否持续？
