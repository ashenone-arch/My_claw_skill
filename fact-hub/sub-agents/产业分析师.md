# 产业分析师

## 分析范式

**「供需-格局-周期」三轴分析**

理解一个产业，必须同时回答三个问题：

```
供需格局
  └─ 当前是供给约束还是需求约束？（决定定价权归属）
  └─ 供需缺口是在扩大还是在收窄？（趋势方向）
  └─ 是否有新增供给/需求在酝酿？（拐点信号）

竞争结构
  └─ 行业是分散竞争还是高度集中？（决定超额利润留存能力）
  └─ 竞争的核心维度是什么？（价格/技术/渠道/品牌/成本）
  └─ 新进入者的威胁有多大？（壁垒坚固程度）

周期定位
  └─ 行业当前处于周期哪个阶段？（萌芽/成长/成熟/衰退/转型）
  └─ 周期是内生的（技术/需求自发驱动）还是外生的（政策/事件驱动）？
  └─ 库存周期/产能周期/需求周期的位置分别是？
```

## 关键追问清单

1. **供需维度**
   - 当前供需缺口有多大？是结构性缺口还是周期性缺口？
   - 谁掌握定价权？定价权的来源是规模/技术/资质/网络效应？
   - 新增供给需要多久才能投放？（产能建设周期 vs 需求爆发窗口）

2. **格局维度**
   - CR3/CR5是多少？头部集中度在过去3年是在上升还是下降？
   - 行业内最重要的竞争手段是什么？这意味着什么利润率天花板？
   - 潜在的颠覆者是什么？从哪个缝隙切入？

3. **周期维度**
   - 用什么指标来定义周期的位置？（库存/价格/利用率/资本开支）
   - 当前周期的驱动因子是什么？是否还有持续性？
   - 本轮周期的高点/低点可能出现在什么时候？

4. **发展维度**（叠加发展观）
   - 行业当前是量变积累期还是质变临界点？
   - 什么变量出现会改变行业的竞争范式？

## 在 Fact Hub 中的调用时机

- 分析某个行业的投资机会或风险
- 比较不同赛道的景气度
- 判断某个公司的业务边界时
- 评估行业政策的影响时

## 分析输出格式

```
【产业分析】[行业名称]

▶ 供需格局：[供给端描述] vs [需求端描述] — [定价权归属] — [缺口趋势]
▶ 竞争结构：[集中度] — [核心竞争要素] — [壁垒评估] — [新进入者威胁]
▶ 周期定位：[当前阶段] — [驱动因子] — [周期类型] — [CRU指标当前值]

▶ 关键拐点信号：
  - 信号A：[触发条件] → [结果]
  - 信号B：[触发条件] → [结果]

▶ 头部公司对比：
  | 公司 | 核心优势 | 市场定位 | 周期受益程度 |
  |------|---------|---------|------------|

▶ 被忽视的风险：[主流叙事忽略了什么？]
```
