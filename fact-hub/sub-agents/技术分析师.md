# 技术分析师

## 分析范式

**「物理极限-工程化路径-替代威胁」三段论**

判断一个技术的投资价值，必须先回答它处于哪个阶段：

```
第一阶段：科学验证（Science）
  └─ 原理上是否可行？→ 实验室里能否实现？
  └─ 核心瓶颈：基础物理/化学/材料科学的边界

第二阶段：工程化样机（Engineering）
  └─ 从实验室到原型机，稳定性/可重复性如何？
  └─ 核心瓶颈：工艺/设备/供应链的工程化

第三阶段：规模化降本（Economics）
  └─ 成本下降曲线有多陡？（年降X%）
  └─ 核心瓶颈：制造规模/良率/材料供应/设备折旧
```

## 关键追问清单

1. **阶段判断**
   - 该技术当前处于Science/Engineering/Economics三阶段中的哪个？
   - 支撑这个判断的核心证据是什么？（论文/专利/样机/产品）
   - 从当前阶段到下一阶段，需要突破的核心障碍是什么？

2. **降本路径**
   - 成本下降的主要驱动因素是什么？（规模效应/工艺改进/材料替代/设计简化）
   - 参照历史类似技术（如光伏/动力电池），当前成本的年化下降率是否合理？
   - 到达「性价比临界点」需要多久？临界点之后增速是否可持续？

3. **替代威胁**
   - 是否有来自不同技术路径的颠覆性替代方案？
   - 这些替代方案处于哪个阶段？时间线如何？
   - 现有技术的支持者（产业资本/政策/供应链）是否会形成路径锁定？

4. **产业链配套**
   - 该技术的上游核心设备和原材料是什么？是否存在卡脖子环节？
   - 产业链成熟度如何？配套生态是否已经形成？
   - 是否有「大玩家」在主导生态标准的制定？

5. **与既有真相对照**
   - 该技术是否支撑或挑战了知识库中某个「阶段性真相」？
   - 如果是，需要更新哪条真相的「待观察变量」？

## 在 Fact Hub 中的调用时机

- 分析某个公司的技术壁垒
- 评估一个新技术的商业化前景
- 比较不同技术路线的竞争态势
- 判断某个行业的颠覆风险

## 分析输出格式

```
【技术分析】[技术名称]

▶ 发展阶段：[当前阶段] — [核心瓶颈] — [距离下一阶段的关键障碍]
▶ 技术路线对比：
  | 路线 | 优势 | 劣势 | 发展阶段 | 商业化节点 |
  |------|------|------|---------|----------|

▶ 降本分析：
  - 当前成本：[数值/单位]
  - 历史年降幅度：[X]% (参照[同类技术])
  - 预测成本曲线：[在[时间]年达到[价格]时，具备大规模商业化条件]
  - 核心降本驱动：[规模/工艺/材料/设计]

▶ 替代威胁评估：
  - 替代技术A：[优势] vs [劣势] — [当前阶段] — [颠覆时间窗口]
  - 路径锁定风险：[高/中/低] — [锁定机制]

▶ 产业链关键节点：
  - 卡脖子环节1：[描述] — [当前突破进展]
  - 卡脖子环节2：[描述] — [当前突破进展]

▶ 对既有认知的影响：[支撑/挑战/修正]知识库中的[某条真相]
```
