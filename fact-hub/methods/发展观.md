# 唯物辩证发展观

## 核心用途

对任何静态事实进行时间维度的解构，避免孤立记录事实。

## 规则模板

### 发展三问

每条事实入库时，必须回答：

1. **它从什么状态演变而来？推动质变的条件是什么？**
   - 回溯：起点是什么？中间经历了哪些量变积累阶段？
   - 识别质变节点：什么条件同时满足时，事物发生了根本性转变？
   - 量变 → 质变不是突变，是积累到临界点后的必然

2. **它当前处于量变积累阶段，还是新旧阶段交替的质变临界点？**
   - 量变阶段：缓慢、渐进、不显著的变化，矛盾双方暂时平衡
   - 质变临界点：矛盾激化、旧平衡打破、新平衡尚未建立的过渡期
   - 判断依据：矛盾的主要方面是否正在发生位移？

3. **未来它内部的矛盾演化，会导致几种可能的发展走向？**
   - 路径一：主要方面继续保持主导，量变延续（维持现状）
   - 路径二：次要方面积蓄力量，发生部分质变（渐进转变）
   - 路径三：矛盾激化，发生根本性质变（新旧交替）
   - 路径N：外部力量介入，改变内部矛盾演化方向（意外转折）

## 发展趋势判断框架

```
当前状态
  ├─ 已知推动力（正向）→ 趋势A
  ├─ 已知阻碍力（负向）→ 趋势B
  ├─ 潜在外部变量         → 趋势C/D
  └─ 临界条件满足度 X%    → 质变概率 = ?
```

## 在 Fact Hub 中的角色

要求每个事实入库时，都附带对其「发展阶段」和「演变动力」的初步判断。知识库中的每条真相，应包含"当前阶段"和"关键观察变量"两个字段。

## 常见分析示例

| 实体 | 当前阶段 | 演变动力 | 关键临界条件 |
|------|----------|----------|-------------|
| 大模型 | 量变积累期 | 算力规模+数据质量 | Scaling law 是否依然有效 |
| 固态电池 | 质变临界点前夜 | 量产工艺突破 | 成本曲线能否在2027年前陡峭下降 |
| 地产行业 | 质变进行中 | 政策力度+出清速度 | 能否在系统性风险阈值前稳住 |
| 消费品牌 | 量变末期 | 品牌心智 vs 价格压力 | 新消费人群的偏好能否形成新量级 |
