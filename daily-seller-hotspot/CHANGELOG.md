# Changelog

All notable changes to this skill will be documented in this file.

## [v1.2] - 2026-05-18

### Changed
- **重构执行模式**：由"完整版（Top20 + 全量溯源）"改为**"速览优先 + 按需溯源"**单流程，默认输出速览版（Top10 + 热点话题 + 核心观点一句话），用户追问时再展开溯源，大幅降低单次执行调用量
- **溯源方式优化**：溯源阶段改为按热度分组打包检索（高热/中热/低热三组），**禁止逐个股串行检索**，解决 Step 4 调用量爆炸问题
- **报告结构简化**：默认速览版不再输出个股详细溯源，信息密度更轻量，适合盘前/盘后快速扫一眼
- **增加行业定向支持**：用户在检索时指定行业关键词（如"半导体"、"AI"），自动过滤相关板块热点

### Fixed
- 第三步从 Top20 降为 Top10，配合溯源按需触发，整体响应速度大幅提升
- 昨日对比默认关闭，用户明确要求时再执行，节省检索成本

## [v1.1] - 2026-05-18

### Changed
- **输出方式**：第六步由"保存为 Markdown 文件 + canvas 预览"改为**直接流式输出到对话中**，无需生成文件，更轻量
- **移除 references/output_format.md**：输出结构直接内嵌于 SKILL.md，不再依赖外部模板文件
- 更新 SKILL.md 描述措辞，使第六步输出方式更明确

### Fixed
- 修正了 iFinD 工具引用方式（从硬编码工具名改为"iFinD MCP 工具"），符合 skill 引用规范

---

## [v1.0] - 2026-04-24

### Added
- 初始版本，支持日度卖方热点选股三段式报告（质检 → 热点个股 Top20 → 个股详细溯源）
- 支持默认"今天"及指定日期查询
- 支持共识强度标签（🔴🟡⚪🔘）
- 支持昨日对比功能（按需）
