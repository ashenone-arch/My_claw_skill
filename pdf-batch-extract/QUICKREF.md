# 快速参考表

## 场景决策

| 场景 | 操作 |
|------|------|
| 用户只给了一个 PDF 文件路径 | 用 read 工具直接读取，无需触发 skill |
| 用户说"批量提取这个文件夹的 PDF" | 触发 pdf-batch-extract skill |
| PDF 有表格要保留 | 自动处理，表格格式化为 Markdown |
| PDF 有水印 | 先询问用户水印内容，再按 WATERMARK.md 清理 |
| PDF 数量 ≤ 2 | 直接执行脚本（单线程） |
| PDF 数量 ≥ 3 | 并行启动 subagent，每个处理 1 个 PDF |
| PDF > 20 页 | 分批读取，每批 10 页 |
| 已有同名 .md 文件 | 提示用户确认是否覆盖，不自动覆盖 |
| 某个 PDF 失败 | 不阻断，继续处理其他 PDF，最终汇总报告 |

## 工具与库

| 工具/库 | 用途 | 命令 |
|---------|------|------|
| `pdfplumber` | 文本提取 + 表格提取 + 页眉/页脚检测 | `pip install pdfplumber` |
| `glob` | 扫描文件夹下的 PDF 文件 | `glob(path, pattern="*.pdf")` |
| `read` | 读取前 3 页检测水印 | `read(file_path, limit=3)` |

## 常见问题处理

| 问题 | 解决方案 |
|------|----------|
| `ModuleNotFoundError: pdfplumber` | 执行 `python -m pip install pdfplumber --quiet` |
| 扫描件 PDF（无文字层） | 提示用户这是扫描件，无法提取文字，建议使用 OCR |
| 密码保护 PDF | 跳过并记录失败，不阻断其他文件 |
| 同名 .md 已存在 | 询问用户是否覆盖，不自动覆盖 |
| 中文路径报错 | Python 3.13+ 原生支持，传绝对路径字符串即可 |
| 表格格式异常 | `extract_text()` 提取的原始文本行作为兜底参考，不做去重 |
| 分批后页眉/页脚遗漏 | 分批读取时复用 detect_headers_footers 结果（一次性识别，复用全场） |

## 水印字符参考

水印文字拆分为单字后，奇数位字（必须清理）和偶数位字（谨慎处理）：

| 位置 | 字符 | 处理 |
|------|------|------|
| 奇数位 | 严、密、保、仅、云、基、参 | 必须清理（行首/行尾/词间） |
| 偶数位 | 格、密、供、锋、金、部、考 | 谨慎处理（仅词间/行尾前字是文本字时） |

```
| 序号 | PDF 文件 | 状态 | 页数 | 表格数 | MD 文件 |
|------|---------|------|------|--------|---------|
| 1 | 文件A.pdf | 成功 | 15 | 3 | 文件A.md |
| 2 | 文件B.pdf | 失败 | - | - | - |
```
